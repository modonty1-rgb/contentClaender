# Content Calendar — Workflow

## الفريق والأدوار

| الشخص | دوره |
|---|---|
| كاتب المحتوى | يكتب الفكرة والمحتوى ويحدد المنصات |
| المصمم | يصمم الصورة / الكاروسيل / الستوري |
| الفيديو إديتور | يكتب السكريبت ويعمل المونتاج |
| الميديا باير | ينشر على المنصات ويتابع الأداء |

---

## حالات التاسك

| الحالة | من يحددها | كيف |
|---|---|---|
| قيد الإنتاج | تلقائي | عند إنشاء التاسك |
| جاهز للمراجعة | المصمم / الفيديو إديتور | بعد رفع الملف |
| جاهز للنشر | كاتب المحتوى | بعد المراجعة والموافقة |
| تم النشر | تلقائي | بعد ما الميديا باير يضيف روابط النشر |

---

## الـ Flow الكامل

```
كاتب المحتوى يكتب التاسك
        ↓
   [تلقائي] Bot → المصمم أو الفيديو إديتور
        ↓
   المصمم/الإديتور يرفع الملف → "جاهز للمراجعة"
        ↓
   [تلقائي] Bot → كاتب المحتوى
        ↓
   كاتب المحتوى يراجع → "جاهز للنشر"
        ↓
   [تلقائي] Bot → الميديا باير
        ↓
   الميديا باير ينشر ويضيف الروابط → "تم النشر" تلقائي
```

---

## المراحل والحقول

### المرحلة 1 — كاتب المحتوى

| الحقل | النوع |
|---|---|
| اليوم | قائمة |
| نوع المحتوى | قائمة |
| مرحلة العميل | اختيار متعدد |
| القنوات | اختيار متعدد |
| الفكرة | نص قصير |
| النص | نص طويل |
| الخطاف | نص قصير |
| الدعوة | نص قصير |
| السكريبت | نص طويل |
| نبرة الصوت | نص قصير |
| الإلهام | نص قصير / رابط |
| ملحوظات | نص طويل |

---

### المرحلة 2 — الإنتاج

**المنطق:**
- صورة / كاروسيل / ستوري → المصمم
- فيديو / ريل → الفيديو إديتور

| الحقل | النوع |
|---|---|
| رابط الملف الجاهز | رابط |

**Action:** يغير الحالة إلى "جاهز للمراجعة"

---

### المرحلة 3 — الميديا باير (صفحة مستقلة)

| الحقل | النوع |
|---|---|
| عضوي / مدفوع | radio button |
| المبلغ | رقم |
| العملة | radio button (ريال / دولار / جنيه مصري) |
| مدة الإعلان | عدد الأيام |
| تاريخ النشر | تاريخ |
| وقت النشر | وقت |
| روابط النشر | per-channel URLs |

**Action:** بعد إضافة الروابط → "تم النشر" تلقائي
